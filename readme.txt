== Description ==
Add/promote RSS feed link to every post. Right now, the rss link appears at the bottom of every post but more customization will be available on request.

*   Add RSS feed link at the bottom of posts.
*   Customize the text as per your need from the options page.

For more information, Please visit [RSS Feed Link to Posts](https://www.inthiscode.com/rss-feed-link-posts).

== Installation ==
Install using Plugins->Add New or manually upload using WordPress plugin uploader.
Activate the plugin, and go to the Settings -> RSS Feed Link to Post.
Check the "**Enable Plugin**" to enable it.
